Game Rules:

    How the Game is played:     < Minesweeper wiki >
  1. Creating a new game will generate a different mine configuration for each time.
  2. You the player can flag squares you believe are mines.
  3. Each flag you place counts against the total mine count.
    • This mean that if you use all of your flags and there are still undiscovered mines you lose.
  4. At any time you can remove a flag previously placed.
  5. Each game will display the number of mines that have been placed, and your corresponding flag choices.
    • Once you have marked all of the mines the game is over and you win!
  6. If you click a square with zero bombs adjacent to it the game will reveal that square as well as any other adjacent square that is not touching a mine.
  7. The game will indicate how many mines are next to a square that has not been revealed.
  8. You the player, can choose the size of your game puzzle and the relative number of mines.
  9. * Remember if a mine is clicked, if all non-mine squares are revealed, or all mine squares are correctly flagged the game will end.
